사용법 안내

마크다운 파일 업로드

  • "노션 또는 파일 업로드" 선택 후 파일 선택 버튼 클릭
  • .md 파일을 직접 선택하거나 드래그 앤 드롭으로 업로드
  • 여러 개의 마크다운 파일을 동시에 업로드 가능

Notion 내보내기 사용법

  • Notion에서 "내보내기" → "마크다운 & CSV" 선택
  • 하위 페이지 포함 옵션 체크
  • 다운로드된 .zip 파일을 직접 업로드
  • zip 파일 내부의 이미지들이 자동으로 처리됩니다

프레젠터 리모컨 지원

  • 로지텍 등 대부분의 USB 프레젠터 리모컨 지원
  • PageUp/PageDown 키로 슬라이드 이동
  • F5 키로 프레젠테이션 모드 시작
  • 레이저 포인터 기능 시뮬레이션 (L 키)
  • 페이지 입력: 하단 입력창에 페이지 번호 입력하여 직접 이동
  • 파일 목록: 여러 파일 업로드 시 파일 간 빠른 전환
  • 로고 클릭: 프레젠테이션 로고 클릭 시 첫 슬라이드로 이동
  • 홈 버튼: 언제든지 메인 화면으로 돌아가기
  • PDF 내보내기: 전체 슬라이드를 PDF로 저장

로컬 파일 이미지

  • 마크다운 파일과 같은 폴더에 이미지 파일 배치
  • 상대 경로로 이미지 참조: ![설명](./image.png)
  • 지원 형식: PNG, JPG, JPEG, GIF, SVG

Notion 이미지

  • Notion 내보내기 시 이미지가 자동으로 zip에 포함
  • 업로드 후 자동으로 Base64로 변환되어 표시
  • 별도 설정 없이 바로 사용 가능
